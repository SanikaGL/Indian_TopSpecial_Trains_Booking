# Train Reservation System

A console-based Train Reservation System built using Python to demonstrate Object-Oriented Programming (OOP) concepts such as classes, objects, methods, and booking logic.

## Features

- View available trains
- Check train details
- View fare information
- Book train tickets
- Track booked and available seats
- Prevent invalid bookings

## Technologies Used

- Python
- Object-Oriented Programming (OOP)

## OOP Concepts Used

This project demonstrates the following OOP concepts:

- Class
- Object
- Attributes
- Methods

### Class Used
- Train

### Methods Used
- `book_ticket()`
- `get_status()`
- `get_fare_info()`

## How the Project Works

1. The program displays available trains.
2. The user can check train details.
3. The user can choose a train and book seats.
4. The system checks seat availability.
5. If seats are available, booking is confirmed and fare is calculated.
